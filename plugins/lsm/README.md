# lsm Claude Code Plugin

Claude Code skills for [lsm](https://github.com/llbbl/lsm).

## Skills

### `/lsm:forensics`

Investigate local `lsm` audit logs using `lsm audit query`, summarize suspicious access patterns, and keep secret values out of the conversation.

## Prerequisites

Install `lsm` and make sure it is available on `PATH`:

```bash
brew install llbbl/tap/lsm
```
